The HTML Renderer Tool

Description:

Hi everybody! This is a test of the html renderer, which is a fantastic tool for creating web pages.

Operations:

A class Html is provided to create a html page. You can create:

  • pages with title and body: use method page
  • paragraphs: use method para
  • lists: use method list
  • items in lists: use method item
  • text elements: use method text
  • line breaks: use method br

Implementation:

Proceed as follows:
  1. First create the classes
  2. Then implement the method render, getTag etc.
  3. Finally test your program with the HtmlApp main class
Good luck!
